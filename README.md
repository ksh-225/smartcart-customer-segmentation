# smartcart-customer-segmentation
SmartCart Customer Segmentation
A minor project that segments e-commerce customers into meaningful groups using unsupervised machine learning.

About
SmartCart collected data of 2240 customers across 22 attributes. Instead of using generic marketing for everyone, this project clusters customers based on their spending behaviour, engagement, and demographics — enabling personalised marketing strategies.

Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook

 
Algorithms Used
-K-Means Clustering

-Agglomerative Clustering

-PCA (for dimensionality reduction & visualisation)

Dataset
smartcart_customers.csv — 2240 records, 22 features including demographics, spending, purchase channels, and campaign response.
